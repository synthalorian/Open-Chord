# 🎹 OpenChord

> **Status:** Ship-ready 🚀 | **Vibe:** 1984 Neon Grid

OpenChord is a high-fidelity guitar chord voicing explorer and fretboard visualizer. Built with Flutter for the modern guitarist who appreciates the aesthetic of the past.

![OpenChord Banner](https://img.shields.io/badge/Aesthetic-Synthwave-ff00ff?style=for-the-badge)
![Flutter](https://img.shields.io/badge/Flutter-3.4+-02569B?style=for-the-badge&logo=flutter)

## ⚡ Features

- **Neon Fretboard:** Dynamic SVG-like rendering with custom glow effects and high-contrast markers.
- **Deep Library:** Supports multiple tunings (Standard, Drop D, DADGAD, Open G, Open D, Half Step Down).
- **Quality Explorer:** Search through Major, Minor, Dominant 7, Major 7, Minor 7, Power Chords, Sus2, and Sus4.
- **Voicing Variations:** Multiple ways to play the same chord, including barre and open positions.
- **Responsive Design:** Optimized for mobile, desktop, and web.

## 🛠️ Tech Stack

- **Framework:** [Flutter](https://flutter.dev)
- **State Management:** [Riverpod](https://riverpod.dev)
- **Storage:** [Hive](https://pub.dev/packages/hive)
- **Typography:** [Google Fonts](https://fonts.google.com) (Orbitron & Inter)

## 🚀 Getting Started

1.  **Clone the grid:**
    ```bash
    git clone https://github.com/synth/open-chord.git
    ```
2.  **Initialize the synthesis:**
    ```bash
    flutter pub get
    ```
3.  **Run the engine:**
    ```bash
    flutter run
    ```
